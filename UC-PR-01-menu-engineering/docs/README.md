## Overview

Menu Engineering Matrix for Luckin Coffee USA — BCG/Kasavana-Smith classification of 58 SPUs (products) across 10 Manhattan stores, with affinity analysis, trend tracking, and an interactive dashboard.

**Key Results:**
- $1.86M cumulative revenue, 560K units sold
- 56.8% revenue-weighted contribution margin
- 14 Stars, 11 Plow Horses, 16 Puzzles, 17 Dogs
- +18.0% monthly order CAGR (Jun 2025 – Jan 2026)
